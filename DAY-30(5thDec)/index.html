BOOTSTRAP

Intro:

  • It is a CSS framework
  • By including some js, we can build responsive web design and mobile-first approach.
  • Bootstrap was developed by Twitter(Creators: Mark Otto, a senior designer, and Jacob Thornton, a developer)
  • It is a frameowrk that contains re-usable components of HTML, CSS and JS.
  • Latest Version: 5.3

Key Features:

  • Responsive Web Design
  • Predefined Components
  • Customizable
  • Cros--browser compatibility
  • Open-source: Anybody can be able to customize it.

Usecases:

  • Single-Page ApplicationsL: Navigation takes place in the same page
  • Multi-Page Applications: Navigation takes place in the new page
  • Hybrid Applications: Combination of both
  • Responsive Web Designs
  • Dashboards
  • E-commerce website etc

Integrating Bootstrap:

  • Dowloading a file
  • Using CDN links
  • npm packages(we can discuss about it in REACT)
