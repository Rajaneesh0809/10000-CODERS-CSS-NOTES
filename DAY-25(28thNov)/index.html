GRID Property in CSS

Through Flex properties we can only alter the either of the dimensions i.e., x-axis or y-axis.

By using Grid Properties we can alter the content for both the dimenions at a time.

  • Grid Container
  • Grid Items
  • Grid template rows
  • Grid template columns

Properties

  • display: grid
  • grid-template-columns: 1fr 1fr (or) 100px 100px 100px (or) repeat(4, 100px) (or) repeat(5, 1fr) -- this property indicates the width of respective column; fr ,means Fractional unit - it basically occupies the free space available
  • grid-template-rows: 150px 250px 50px (or) 3fr 5fr 1fr-- this property indicates the height of the respective row;
  • grid-template-columns: repeat(auto-fill, minmax(200px, 1fr))
  • row-gap: 10px
  • column-gap: 10px
