Flex-Box Model in CSS

Properties:

  • flex-direction: row, column, row-reverse, column-reverse
  • flex-wrap: wrap, nowrap
  • justify-content: center, space-around, space-evenly, space-between
  • align-content: center, start, end
  • align-items: center, start, end

NOTE: Explore all the values for these properties.

Flex Item properties

These properties are used only for the items which are inside the flex-box, not for the flex-box.

  • order-
    • the item with higher order stays at the right(ascending order)
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self
